# Arduino

## Начнем с теории
Arduino - 
<img src="https://github.com/mook003/Triados/blob/main/docs/images/arduino.jfif">

Из-за особенностей архитектуры Nvidia Jetson Nano управление ШИМ-сигналами ограничено. У компьютера присутствует лишь 2 соответствующих пина.
В качестве посредника в передаче сигналов на двигатели и сервоприводы был выбран микроконтроллер Arduino (не важно какая модель микроконтроллера, принцип одиноаковый), ведь у него присутсвует необходимое количество пинов и для взаимодейсвия с двигателями, и с сервоприводами.
